# TrapScuro — Shopify OS2.0 theme

Tema Shopify Online Store 2.0 (stile Dawn) per streetwear “TRAP SCURO”: minimal, dark, high-contrast, CTA nette.

## Branding quick notes

**Palette**
- Nero: `#000000`
- Antracite: `#1A1A1A` / `#222222`
- Accento (rosso scuro): `#5A0B0B` / `#6B0F0F`

**Tone of voice**
- diretto, underground, senza fronzoli (es. “Drop”, “Limited”, “Size & Fit”, “Non dormire”).

## Linee guida immagini prodotto
- Sfondo: nero/grigio opaco uniforme (coerenza feed).
- Luce: rim light laterale + fill morbido frontale; controlla highlight su stampe.
- Scatti: front/back + dettaglio stampa + fit shot.
- Lookbook: location urbana minimale, scatti notturni, accento rosso (neon/lampade).
- Output: 1:1 e 4:5, minimo 2048px lato lungo, neri profondi ma con texture (no “crush” totale).

## Note tecniche
- Theme root è la root del repository (cartelle tema: `assets/`, `config/`, `layout/`, `locales/`, `sections/`, `snippets/`, `templates/`).
- `config/settings_data.json` non è versionato (ambiente-specifico).
